What is Linear Regression


Linear regression is a linear approach for modeling the relationship between a dependent variable y and one or more independent variables denoted X.The overall idea of regression is to examine three things:

  1. Does a set of predictor variables do a good job in predicting an outcome variable?
  2. Which variables in particular are significant predictors of the dependent variable?
  3. What is the regression equation that shows how the set of predictor variables can be used to predict the outcome?

Uses of Linear Regression
Identify the strength of the effect that the independent variable(s) have on a dependent variable Forecast effects or impact of changes Regression analysis predicts trends and future values
Typical Question: What is the strength of relationship sales and marketing spend? Typical Question: How much additional Y do I get for one additional unit X? Typical Question: What will the price for gold be in 6 month from now?

Different Types of Linear Regression
Type Description
Simple linear regression 1 Dependent Variable + 1 Independent Variable
Multiple linear regression 1 Dependent Variable + 2+ Independent Variables
Logistic regression 1 Dependent Variable (binary) + 2+ Independent Variable(s)
Ordinal regression 1 Dependent Variable (ordinal) + 1+ Independent Variable(s)
Discriminant analysis 1 Dependent Variable (nominal) + 1+ Independent Variable(s)
